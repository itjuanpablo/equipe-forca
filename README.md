# 🌐 README — Portal Equipe Força

## 📌 Descrição

Este projeto é um portal web institucional da **Equipe Força**, desenvolvido em HTML, CSS e JavaScript.
O site tem como objetivo conectar jovens, divulgar células (pequenos grupos), recrutar voluntários e facilitar a inscrição em eventos como a *Revisão de Vidas*.

---

## 🚀 Funcionalidades

### 🏠 Página Inicial

* Apresentação da Equipe Força
* Versículo bíblico inspiracional
* Botão de ação para participação em células
* Links para redes sociais (Instagram)

---

### 📍 Células

* Lista completa de células organizadas por:

  * Nome
  * Bairro
  * Dia e horário
* Links diretos para localização no Google Maps
* Links para contato via Instagram dos líderes

---

### 🤝 Voluntariado

* Seção para recrutamento de voluntários
* Áreas disponíveis:

  * Mesa de som e transmissão
  * Stories e fotos
  * Projeção
* Contato direto via WhatsApp

---

### 🎧 Spotify

* Playlist integrada via embed do Spotify
* Foco em músicas utilizadas nos encontros (ex: Arena One)

---

### 💙 Revisão de Vidas

* Apresentação do evento
* Contato com líderes para inscrição via:

  * WhatsApp
  * Instagram
* Download da ficha de inscrição em PDF

---

### 📱 Contato e Redes Sociais

* Links rápidos para Instagram oficial
* Contato direto com líderes
* Navegação simples e intuitiva
